# Web Development Practice Projects

This repository contains a collection of web development practice projects created as part of a web development learning path.

The project is being developed by **Mohammadali Mousavireineh** together with one of his students, who has completed the web development course. The main goal of this repository is to practice real web development skills by building small experimental websites and improving them step by step.

## About This Repository

This repository is not a single final product. It is a practical workspace for learning, practicing, and improving front-end and basic web development skills.

We use this repository to build and test different website ideas, layouts, components, and user interface designs.

## Goals

The main goals of this project are:

* Practice HTML, CSS, and JavaScript
* Build small experimental websites
* Improve layout and responsive design skills
* Practice clean project structure
* Learn how to work with Git and GitHub
* Improve teamwork between instructor and student
* Prepare for larger real-world web development projects

## Technologies Used

* HTML

* Node.js

## Getting Started

To run this project locally, follow these steps.

### 1. Clone the repository

```bash
git clone https://github.com/mohammadali-mousavireineh/webdevelopment.git
```

### 2. Go to the project folder

```bash
cd webdevelopment
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open the project in your browser

```text
http://localhost:1000
```
